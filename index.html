  4. :empty選擇的元素裡面沒有任何內容。

3.減少不必要的html標籤

偽元素(-Wiki)

虛擬類別元素(-CSS3 樣式設計手冊)

以假定性html標籤製作出包圍後的元素

偽元素 就像是 火影忍者

鳴人的影分身之術，使他能夠一個人施展螺旋丸。

HTML

CSS	
div.naruto1:before,div.naruto1:after{
	忍術: 影分身;
}

div.naruto1:before,div.naruto1:after{
